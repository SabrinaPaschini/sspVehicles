# SSP Vehicles

Sistema web para gerenciamento de veículos desenvolvido com ASP.NET MVC, C#, SQL Server e Dapper.

## Sobre

Este projeto tem como objetivo consolidar conhecimentos em desenvolvimento backend, arquitetura em camadas, banco de dados e boas práticas de desenvolvimento.

O sistema permitirá o gerenciamento de fornecedores, veículos, estoque e vendas, além da integração com a API FIPE para consulta de informações dos veículos.
